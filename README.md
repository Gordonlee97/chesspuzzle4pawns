# Four Pawns

A chess puzzle built with React + TypeScript.

**[Play it here](https://gordonlee97.github.io/chesspuzzle4pawns/)**

## The puzzle

You have a knight, rooks, bishops, and pawns. The four black pawns are on the other side of a wall. Capture all four of them to win.

Move pieces by clicking or dragging. Click a selected piece again to deselect it. Use undo if you need to backtrack. The move counter tracks how many moves you've taken — try to beat your best score.

## Running locally

```bash
npm install
npm run dev
```

## Tech

- React 19, TypeScript, Vite
- Pure CSS — no UI libraries
- Drag-and-drop via mouse events — no libraries
- Deployed to GitHub Pages via GitHub Actions
